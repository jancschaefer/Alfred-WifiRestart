# WiFi-Restart

This is a workflow for [Alfred](http://alfredapp.com) that restarts your wifi connection in case it got stuck (eg. due to a VPN Issue.)

## Usage
Simply Install the ```.alfredworkflow``` file and use the keyword ```wifirestart```
